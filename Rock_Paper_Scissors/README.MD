# Simple Rock Paper Scissors 🪨

This is a simple command-line rock paper scissors program written in C. It allows you to play your childhood game.

## Features

- Infinity gameplay
- Scoreboard.
- Simple colorful interface.
- Exit option.

## Prerequisites

Before you can use this game, you need to have the following:

- A C compiler (e.g., GCC) installed on your system.

## Usage

1. Compile the source code by running the following command:

   ```shell
   make
    ```
2. Run the calculator program by executing the `rockpaperscissors` executable.

   ```shell
   ./bin/rockpaperscissors
   ```

3. You will see the menu with the available options:

   ```
    User commands
     1. Rock
     2. Scissors
     3. Paper
     4. ScoreBoard
     5. Exit
   ```

4. Enter the number corresponding to your desired operation:

   - For rock, enter `1`.
   - For scissors, enter `2`.
   - For paper, enter `3`.
   - For scoreboard, enter `4`.
   - To exit the program, enter `5`.
